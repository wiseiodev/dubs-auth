# Shareable Skills

Public, installable skills for this repository live under `skills/`.

## Install

```bash
npx skills add wiseiodev/dubs-auth/skills/dubs-auth-skill
```

## Available

- `dubs-auth-skill`: End-to-end guidance for integrating
  `@wiseiodev/dubs-auth` in Next.js App Router apps with Better Auth +
  Stripe + Drizzle.
